《切韻》《玉篇》聯合查詢 - 使用説明

收録資料

《玉篇》系： 原本《玉篇》、《玄應音義》、宋本《玉篇》、《篆隸萬象名義》、《新撰字鏡》、《類聚名義抄》、《玉篇》佚文（澤田達也）

《切韻》系：唐五代《切韻》系韻書、《廣韻》

查字功能

【輸入要求】請輸入繁體中文；簡化漢字無法返回。

【異體字】默認關聯，也可以點擊下拉菜單選擇不關聯。

【正查or反查】默認查字頭，也可以點擊下拉菜單，反查注文。

【顯示全部or正文】默認顯示全部，也可以點擊下拉菜單，只查正文。這個功能是爲寫論文方便複製設計的。

資料來源

- 原本《玉篇》：來自一個已經關閉的 GitHub 項目，比小鏡更全。
- 《萬象名義》、《新撰字鏡》、宋本《玉篇》、《類聚名義抄》：來自 HDIC。
- 《玉篇》佚文：來自澤田達也先生項目。
- 《切韻》諸本：來自鈴木慎吾先生《切韻諸本輯覽》。
- 《廣韻》：源數據來自NK2028。批量用賢超法師的工具進自動標點處理，訛誤很多。
【附帶談談】
- 所有數據來源於網絡，錯録、誤録之處難以避免，正式引用時請核對原書。
- 宋本《玉篇》和《廣韻》都是澤存堂本。澤存堂刻書好改字，已距宋刻很遠。嚴肅研究應核對兩書的寧宗原版。
